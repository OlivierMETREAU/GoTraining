# Day 2 - Minimal HTTP Server

## Objective
Create a simple HTTP server with JSON endpoints.

## Tasks
- Create a server using net/http.
- Endpoints:
  - /hello
  - /time
  - /status
- Add a logging middleware.
- Organize code into packages.

## Go Concepts
HTTP, handlers, JSON, modular code.

## Extra Exercises
- Add /metrics.
- Use a third-party router (chi, gorilla/mux).
- Test a handler using httptest.
