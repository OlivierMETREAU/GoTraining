# Day 4 - Plugin System (Interfaces)

## Objective
Create a system of interchangeable modules using interfaces.

## Tasks
- Define a Processor interface.
- Implement 3 modules:
  - Uppercase
  - Lowercase
  - ROT13
- Select module via CLI argument.
- Apply module to a string.

## Go Concepts
Interfaces, polymorphism.

## Extra Exercises
- Add a reverse module.
- Add a leet-speak module.
- Load modules from a config file.
