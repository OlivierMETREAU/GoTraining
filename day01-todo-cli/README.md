# Day 1 - Todo CLI

## Objective
Create a CLI application to manage a todo list.

## Tasks
- Create a Task struct (id, description, done).
- Read/write a JSON file (tasks.json).
- Implement commands:
  - add "text"
  - list
  - done <id>
  - delete <id>
- Use the flag package to parse arguments.
- Organize the code cleanly.

## Go Concepts
Structs, slices, JSON, file handling, CLI.

## Extra Exercises
- Add a creation date.
- Add a search command.
- Add CSV export.
