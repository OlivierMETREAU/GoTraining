# Day 3 - Log Analyzer

## Objective
Analyze a log file and produce statistics.

## Tasks
- Read a file line by line.
- Extract IP, date, endpoint, etc.
- Use a map to count occurrences.
- Generate a text report.
- Add a --json output mode.

## Go Concepts
Maps, parsing, file handling.

## Extra Exercises
- Detect most frequent IPs.
- Generate an ASCII histogram.
- Add a live mode (tail -f).
