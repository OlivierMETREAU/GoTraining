# Day 9 - TCP Chat

## Objective
Create a TCP server allowing multiple clients to chat.

## Tasks
- Create a TCP server.
- Handle multiple clients via goroutines.
- Broadcast messages.
- Handle disconnects.
- Add nicknames.

## Go Concepts
TCP, goroutines, channels.

## Extra Exercises
- Add message history.
- Add commands (/who, /quit, /nick).
- Add a minimal GUI client.
