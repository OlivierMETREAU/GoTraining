# Day 5 - Concurrency Pipeline

## Objective
Create a concurrent processing pipeline.

## Tasks
- Read a list of numbers.
- Step 1: filter even numbers.
- Step 2: multiply by 3.
- Step 3: sum results.
- Each step runs in a goroutine.
- Use channels for communication.

## Go Concepts
Goroutines, channels, concurrency patterns.

## Extra Exercises
- Add a timeout using context.
- Add a worker pool.
- Add a verbose mode.
